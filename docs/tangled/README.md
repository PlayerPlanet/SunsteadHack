# Tangled Track — Exploration Docs

Exploration docs for the **governance + explainability** track, built on Tangled
(decentralized git on the AT Protocol). This track is the *generalization* of the Aiven
autoresearch track (see `../solution-directions.md`): where Aiven has an objective metric
that needs no human judge, this track is about **governed, accountable judgement** for the
surfaces that don't.

These are **exploration** docs — they capture the design space and the open questions, not a
finalized spec. Edit freely.

| Doc | Purpose |
|-----|---------|
| [01-concept.md](01-concept.md) | What the track is, the thesis, how it relates to the Aiven track |
| [02-atproto-data-model.md](02-atproto-data-model.md) | What we've **confirmed** about Tangled/ATProto (with citations + caveats) |
| [03-lexicon-design.md](03-lexicon-design.md) | Draft custom lexicons (`com.sunstead.*`) for crossings/judgments/outcomes |
| [04-open-questions-and-spikes.md](04-open-questions-and-spikes.md) | Unknowns + the spikes to run to de-risk, with pass/fail criteria |

## Status

Exploration only. **No code committed for this track yet.** The Aiven autoresearch loop is the
current front-runner for the submission; this track is being de-risked in parallel because its
ATProto integration carries the most uncertainty.
