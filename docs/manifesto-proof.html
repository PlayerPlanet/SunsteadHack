READING 01

The autonomous interior works.

The loop proposes a change, applies it to a real Aiven database, measures it on a frozen benchmark, and keeps-or-rolls-back. On one workload it cut p99 from a sequential scan to an indexed lookup — and refused a change that didn't beat the new baseline. The descending curve is trustworthy because the optimizer proposes and a separate frozen judge scores.

READING 02

It stops, and asks.

When an action would be unsafe to be wrong about, a frozen, dumb pore refuses to auto-apply and escalates. A human decides; the decision is written back. This trail ran live across four separate processes sharing only the Aiven log — the operator drove every step from a Claude session, and Claude never scored an experiment.

  1. Define a taskRegistering a new objective is a define action — governed, not free.
  2. The pore stops itFrozen rule reversibility fired → requires_human_judgment. Written as a crossing.
  3. A human adjudicatesApproved with rationale → judgment row, judge_kind = human.
  4. The task activatesOnly now is it dispatchable. Every step is an immutable row in the escalation log.
READING 03

The boundary, measured — not asserted.

Two readings drawn straight off the log. Spatial: as the workload drifts from what the system has earned trust on, the agent's own proposals cross the frozen pore more often — escalations rise. Longitudinal: at a fixed workload the line is flat by design — the frontier that does not yet recede.

Proxy

The shippable pore gates blast-radius + reversibility — "would this be safe to be wrong about." That lower-bounds, but is not, the true epistemic edge. We label it a proxy everywhere, and never mistake it for the boundary itself.

The honesty ledger

Proven

The autonomous interior + the stop-and-ask + the schema'd escalation log + the instrument. Descending p99 on live Aiven; a human judgment recorded across processes; both boundary readings drawn off the log.

The bet

A calibrated, OOD-aware membrane that bends the longitudinal line down — the frontier receding faster than ambition reopens it. Claiming this as a result would be the very failure the thesis warns against. We measure it; we do not fake it.
